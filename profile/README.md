# Regole Repo

Per mantenere ordine, uniformità e facilitare la ricerca all'interno dell'organizzazione, è strettamente obbligatorio rispettare le seguenti convenzioni per la nomenclatura dei repository:

* **Repository di Progetto:** Ogni repository relativo a un progetto principale o a un'esercitazione pratica deve iniziare con il tuo cognome seguito dal nome del progetto.
  * **Formato:** `cognome-nomeprogetto`

* **Esercizi Extra e Teoria:** Qualora il repository sia destinato ad accogliere esercizi aggiuntivi, appunti o prove pratiche legate alla teoria, la nomenclatura deve includere il tuo cognome, la parola "teoria" e la materia specifica di riferimento.
  * **Formato:** `cognome-teoria(materia)`
